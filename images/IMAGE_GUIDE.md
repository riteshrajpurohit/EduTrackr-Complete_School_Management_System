# Images Setup Guide

The README now has a beautiful screenshot gallery section. Please place your 5 screenshots in this folder with the following names:

## Required Images

1. **01-login-screen.png**
   - Screenshot of the login/authentication portal
   - Shows role selection and Google OAuth option
   - Recommended size: 1200x800px or similar

2. **02-fees-management.png**
   - Screenshot of the Fees Management Dashboard
   - Shows "Manage Fee Groups", "Manage Installments", "Manage Extra Fees", etc.
   - Recommended size: 1200x800px or similar

3. **03-principal-console.png**
   - Screenshot of the Principal/Super Admin Console
   - Shows dashboard with classes, subjects, teachers, and students metrics
   - Recommended size: 1200x800px or similar

4. **04-student-attendance.png**
   - Screenshot of Student Attendance tracking page
   - Shows attendance records, percentage, and history
   - Recommended size: 1200x800px or similar

5. **05-student-dashboard.png**
   - Screenshot of Student Welcome/Home Dashboard
   - Shows class info, marks, and quick action links
   - Recommended size: 1200x800px or similar

## How to Add Images

1. Save or export each screenshot as PNG format
2. Rename them according to the names above
3. Move them to this `images/` folder
4. The README will automatically display them in the gallery

## Image Format Tips

- **File Type**: PNG or JPG
- **Dimensions**: 1200x800px (or similar widescreen ratio)
- **Quality**: Clear and readable (at least 72 DPI)
- **File Size**: Keep under 500KB per image for faster loading

Once images are placed, the README gallery will display beautifully! ✨
